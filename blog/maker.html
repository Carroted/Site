Welcome to the blog post maker for Carroted members or guest writers!

If you aren't a Carroted member or invited writer, this might not be of much use to you, unless you're making your own blog using the Carroted Blog code as a base.

Please fill out the following fields with your article's info and content! You can use an HTML editor if you want, such as https://html-online.com/editor/, or just manually write HTML for extra customization.

Once done, just send the downloaded file to me (Alex) on Discord! I'll add it to the blog.

Enjoy!
